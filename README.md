# WA Baileys Bot - Render

## Setup
1. Edit `NOMOR_TUJUAN` di `index.js`
2. Tambah/kurangi `COMMANDS` sesuai kebutuhan
3. Push ke GitHub → connect ke Render
4. Lihat log Render → scan QR code yang muncul

## Cara pakai
- Kirim pesan **"mulai"** ke nomor bot ini
- Bot otomatis kirim command satu per satu ke nomor tujuan
- Setiap kali nomor tujuan membalas, bot kirim command berikutnya
